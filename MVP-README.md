# 亲密关系依恋 & 界限感综合测评 - MVP版本

## 🎯 项目简介

这是一个基于成人依恋理论的心理测评Web应用，帮助用户了解自己在亲密关系中的依恋模式、界限感和自我价值状况。

## 📦 项目结构

```
├── index.html          # 首页
├── quiz.html           # 答题页
├── unlock.html         # 兑换码验证页
├── report.html         # 报告页
├── css/
│   ├── style.css       # 全局样式
│   ├── landing.css     # 首页样式
│   ├── quiz.css        # 答题页样式
│   ├── unlock.css      # 解锁页样式
│   └── report.css      # 报告页样式
├── js/
│   ├── quiz.js         # 答题逻辑
│   ├── algorithm.js    # 核心计算算法
│   ├── unlock.js       # 兑换码验证
│   └── report.js       # 报告生成
└── data/
    ├── questions.json  # 题库数据（24题）
    └── content.json    # 文案数据
```

## 🚀 快速开始

### 本地运行

1. 直接在浏览器中打开 `index.html`
2. 或使用本地服务器运行：

```bash
# 使用 Python
python -m http.server 8000

# 使用 Node.js (需要全局安装 http-server)
npx http-server -p 8000
```

3. 访问 `http://localhost:8000`

## 📊 核心功能

### 1. 答题系统
- 24道题目，5级量表评分
- 自动跳题，进度保存
- 支持返回修改答案

### 2. 核心算法
- **四维度评分**：依恋焦虑（A）、依恋回避（B）、界限感（C）、自我价值脆弱度（D）
- **等级判定**：低/中/高
- **类型判定**：7种依恋类型（安全型、焦虑型、回避型、恐惧-回避型、偏焦虑混合型、偏回避混合型、混合复杂型）

### 3. 可视化报告
- 四维雷达图
- 得分条展示
- 详细文案解读
- 界限感和自我价值分析

### 4. 兑换码系统
- 输入兑换码解锁完整报告
- 未解锁时显示基础评分
- 支持跳过查看基础版

## 🎨 设计风格

- **配色方案**：温暖治愈的暖粉色系
- **字体**：Noto Sans SC（思源黑体）
- **交互**：流畅的动画过渡效果
- **响应式**：移动端优先设计

## 📝 数据存储

MVP阶段使用 LocalStorage 存储数据：
- `answers`：用户答题记录
- `result`：计算结果
- `unlocked`：是否已解锁完整报告

## 🔧 技术栈

- **前端**：纯HTML/CSS/JavaScript（无框架）
- **图表**：Chart.js
- **字体**：Google Fonts
- **存储**：LocalStorage
