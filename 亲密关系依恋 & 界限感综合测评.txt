## 一、测评基本设定

- **名称**：亲密关系依恋 & 界限感综合测评
- **题目数量**：24 题
- **适用对象**：有过或正在经历亲密关系 / 暧昧关系的成年人
- **作答范围**：请根据**最近 6 个月在“亲密/暧昧关系”中的真实感受**作答
- **作答方式（5 级量表）**：
   1 = 非常不同意
   2 = 比较不同意
   3 = 说不清 / 一般
   4 = 比较同意
   5 = 非常同意
- **四个维度**：
  1. **依恋焦虑（A）**：怕被抛弃、怕失去、过度担心对方
  2. **依恋回避（B）**：害怕亲密、习惯抽离、防御性独立
  3. **界限感健康度（C）**：是否能说“不”、区分自己和他人的责任
  4. **自我价值脆弱度（D）**：在亲密关系里自我价值感有多容易被动摇

## 三、计分逻辑（给你或程序员看的）

### 1. 基本计分

- 每题原始分 = 1–5
- **反向计分题**：
  - 公式：反向分 = 6 - 原始分
  - 反向题号：5、6、11、12、14、16、18

### 2. 各维度得分

> 建议最终使用“平均分（保留一位小数）”，方便直观展示 1–5 分。

- **依恋焦虑 A_score**
  - 题号：1,2,3,4,5(反向),6(反向)
  - A_sum = 上述 6 题换算后的分数之和
  - A_avg = A_sum / 6
- **依恋回避 B_score**
  - 题号：7,8,9,10,11(反向),12(反向)
  - B_sum，同理
  - B_avg = B_sum / 6
- **界限感健康度 C_score**
  - 题号：13,15,17 正向；14,16,18 反向
  - C_avg = C_sum / 6（分数越高 = 界限越清晰、越健康）
- **自我价值脆弱度 D_score**
  - 题号：19–24 全部正向
  - D_avg = D_sum / 6（分数越高 = 自我价值越脆弱、越容易因关系波动而否定自己）

------

### 3. 维度等级划分（Low / Mid / High）

> 可以在前端直接以“低/中/高”+进度条展示。

建议统一阈值（你也可以根据实际样本微调）：

- **低**：avg ≤ 2.4
- **中等**：2.5 ≤ avg ≤ 3.5
- **高**：avg ≥ 3.6

对每个维度得到一个等级标签，比如：

- A_level ∈ {低、中、高}
- B_level ∈ {低、中、高} ……

------

### 4. 依恋类型判定逻辑

用 A（焦虑）和 B（回避）的“低/中/高组合”来判定：

1. **安全型依恋**
   - 条件：A_level = 低 且 B_level = 低
2. **焦虑型依恋**
   - 条件：A_level = 高 且 B_level ≠ 高（低或中）
3. **回避型依恋**
   - 条件：B_level = 高 且 A_level ≠ 高（低或中）
4. **恐惧-回避 / 矛盾型依恋**
   - 条件：A_level = 高 且 B_level = 高
5. **混合 / 过渡型**（给系统一个兜底）
   - 其他情况：
     - 若 A_avg > B_avg → 输出“偏焦虑型”
     - 若 B_avg > A_avg → 输出“偏回避型”
     - 若两者差值 ≤ 0.3 → 输出“混合型 / 难以归类”，文案更温和说明“你的依恋模式比较复杂，常在不同情境下切换”。

### 4.2 各依恋类型详情模板

下面是 4 个主类型 + 2 个扩展类型的文案框架。
 你可以按“段落结构 + 要点”来写长版，也可以直接用我给的示例句。

------

#### A. 安全型依恋（A 低，B 低）

**1）核心画像**

> 你在关系中整体是**安全型依恋**。
>  你能够在亲密和独立之间找到比较舒服的平衡：既愿意亲近、也允许彼此有空间。你会相信自己值得被爱，也基本相信对方是可靠的。

**2）典型表现**

- 能够比较坦诚地表达需要和情绪
- 冲突出现时，更愿意沟通而不是冷战 / 消失
- 不太会因为对方短暂的冷淡就完全否定自己
- 能接受“对方有情绪 ≠ 我做错了什么”

**3）潜在盲区**

> 即便是安全型，在压力大或遇到特殊对象时，也可能出现阶段性的焦虑 / 回避反应。

- 有时可能会低估对方的不安全感，而显得“太理性”
- 容易在与高度不安全型伴侣相处时，被拖入消耗模式

**4）成长建议**

- 继续保持“愿意沟通 + 接受差异”的态度
- 面对极度不稳定的关系时，也要把注意力放在“是否尊重我的边界”，而不是自我怀疑
- 可以尝试把你稳定、成熟的相处方式，用语言清晰地说出来，帮助对方学习安全感

------

#### B. 焦虑型依恋（A 高，B 非高）

**1）核心画像**

> 你在关系中更接近**焦虑型依恋**。
>  你对亲密、陪伴和确认有很强的需求，一旦感受到对方的疏离或冷淡，你会非常敏感，容易陷入“是不是要被抛弃了”的担心。

**2）典型表现**

- 非常在意对方的态度变化，对消息回复速度、语气高度敏感
- 常常会在心里预演最坏情况，例如“他是不是不爱我了”
- 冲突发生时，倾向于追着对方确认、解释、讲清楚
- 很容易把对方的情绪归因到“是我不够好 / 我又做错了”

**3）可能的风险**

- 在关系里过度付出、低估自己需要，换来的是越来越不平衡
- 情绪起伏大，影响工作、睡眠甚至其他人际关系
- 容易吸引“回避型”伴侣，形成拉扯剧本：你越追，对方越躲

**4）具体建议**

- 练习分清楚：
  - “对方此刻很冷淡” ≠ “他永远不爱我了”
- 把注意力从“猜他的想法”，转移到“看他的长期行为”
- 先学会在关系外建立支撑系统：朋友、兴趣、工作成就感
- 可以结合测评中【界限感 / 自我价值】结果，重点练习：
  - 不用靠过度付出证明自己的价值
  - 不用每一次情绪波动都立刻求确认，允许关系有一点点不确定性

------

#### C. 回避型依恋（B 高，A 非高）

**1）核心画像**

> 你在关系中更接近**回避型依恋**。
>  你对亲密有渴望，但更本能的反应是“保持距离”。一旦感觉到对方太黏、太情绪化，你就想后退、冷静、甚至直接消失。

**2）典型表现**

- 不习惯在别人面前展示脆弱，更倾向于自己消化
- 当对方情绪强烈或要求太多时，会觉得被“控制 / 压迫”
- 害怕给出承诺，因为一旦答应，就会觉得自己被绑住了
- 经常用工作、兴趣、自己一个人的时间，来维护安全感

**3）可能的风险**

- 让伴侣持续处于不安状态，对方容易认为你“不爱 / 不在乎”
- 长期压抑自己的需求，反而会在某个节点一下子爆发、想逃离
- 容易进入“要么很近，要么完全断联”的极端状态

**4）具体建议**

- 把“亲密 = 被控制”改写成：“亲密 = 有人可以一起扛事”
- 允许自己从小步开始分享：每天只多说一点点真实感受即可
- 当你想退缩时，提前和对方约定：“我现在需要一点自己的时间，XX 点之后我会回来和你聊。”
- 在保证界限感的前提下，多练习“向他人求助”，这是对你自己的温柔，而不是软弱

------

#### D. 恐惧-回避 / 矛盾型（A 高，B 高）

**1）核心画像**

> 你在关系中更接近**恐惧-回避 / 矛盾型依恋**。
>  你既非常渴望亲密，又非常害怕被伤害。靠近时会紧张、不安，远离时又会强烈想念，对自己和对方都容易产生怀疑。

**2）典型表现**

- 有时像典型的焦虑型：很需要确认，很怕失去
- 有时又特别回避：突然冷淡、拉开距离、不想面对问题
- 对自己的评价摇摆大：一会儿觉得自己很糟，一会儿又觉得对方才有问题
- 往往有比较复杂的过往经验：比如被忽视、被突然抛弃、关系多次“翻车”等

**3）可能的风险**

- 关系的节奏反复拉扯：靠近——害怕——逃离——后悔——再靠近
- 很难真正相信“有人会稳定地在我身边”，容易陷入反复试探
- 对情绪和精力消耗都很大

**4）具体建议**

- 不要一个人硬扛，可以考虑更系统地学习 / 咨询（如果有条件）
- 尝试把“我现在在怕什么”说清楚，而不是只表现为冷淡或抓紧
- 提前和对方约好“吵架 / 情绪波动时的安全规则”
- 你可以从提升【界限感】和【自我价值稳定度】两块入手，把关系当成练习场，而不是考试场

------

#### E. 偏焦虑型（混合 / A 略高于 B）

文案思路：

- 先说明你同时带有焦虑 & 回避特征
- 再强调“在不安全时更容易偏向焦虑一侧”
- 建议参考焦虑型的建议，并加上一条“注意别在害怕时突然抽离”

你可以这样写：

> 你的依恋模式中，同时带着焦虑和回避的特征，但在关键时刻更容易**偏向焦虑一侧**。
>  …（后续参照焦虑型，只是语气柔和一点，强调“不是固定标签，而是一种倾向”）

------

#### F. 偏回避型（混合 / B 略高于 A）

同理，主要参考回避型文案，只是加一句：

> “在感觉安全的时候，你也会主动靠近和付出。”

### 4.3 界限感健康度（C）文案模板

按高 / 中 / 低给三套模板：

**1）C 高（界限感较健康）**

> 你在关系中的界限感整体比较健康。
>  你能区分“我想要的”和“对方期待我做的”，在不舒服时也愿意表达。这样的你，更容易在亲密的同时保持自尊和自主。

**2）C 中（界限感有时模糊）**

> 你的界限感处在一个“忽好忽弱”的状态。
>  当对方态度温和时，你能讲出自己的想法；但一旦对方情绪变强，你就容易退让、迁就。你其实知道自己想要什么，只是还在练习“把它稳定地说出来”。

**3）C 低（界限感较薄弱）**

> 你在关系中容易为了维持和谐，而牺牲自己的感受。
>  经常会答应一些自己并不想做的事，也常把责任往自己身上揽。长期下来，会累积很多委屈和内耗。
>  这并不是你“软弱”，而是你习惯把“对方不生气”放在自己之前。
>  建议在日常小事里先练习说“不”：从不太重要的请求开始，一点点把自己放回中心。

------

### 4.4 自我价值脆弱度（D）文案模板

**1）D 高（自我价值较脆弱）**

> 在亲密关系里，你的自我价值感比较容易被动摇。
>  一点点拒绝、冷淡，都会触发“是不是我不够好”的想法。
>  这种模式往往来自过去一次次的受伤经历，并不是你的错。
>  接下来，你可以尝试把注意力从“我哪里不够好”，转向“我是否在好好对待自己、选择对我友善的人”。

**2）D 中（自我价值有波动）**

> 你的自我价值感整体还可以，但在特别在意的人面前会明显变得脆弱。
>  情绪好的时候，你觉得自己也挺不错；情绪低的时候，就容易怀疑自己一无是处。
>  你需要的不是“变得完美”，而是学会在自我评价低落时，仍旧做一些照顾自己的决定。

**3）D 低（自我价值较稳定）**

> 在关系中，你对自己的整体评价比较稳定。
>  即便遇到冲突或被拒绝，你也能把那理解为“双方不合适 / 事情本身的问题”，而不是简单地等于“我不行”。
>  这份稳定，是你在关系里的底气，也能让你更有空间照顾到对方的情绪。
